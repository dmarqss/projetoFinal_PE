# Projeto de Exercícios Interativos em C

Este é um projeto desenvolvido em C, que oferece uma série de funções e exercícios interativos baseados em cálculo, verificações, e desenhos geométricos. O programa exibe um menu com diversas opções, onde o usuário pode escolher qual exercício deseja realizar. O projeto inclui funções de cálculo de salários, verificação de ano bissexto, conversões de temperatura, entre outras.

## Funcionalidades

O programa oferece uma variedade de funcionalidades divididas em diferentes categorias. Algumas das principais funcionalidades incluem:

- **Cálculos:**
  - Calcular peso ideal
  - Converter temperatura entre Celsius e Fahrenheit
  - Calcular salário com base em comissão
  - Calcular soma de números
  - Calcular média de notas
  - Calcular economia anual

- **Verificações:**
  - Verificar se um número é positivo, negativo ou zero
  - Verificar se um ano é bissexto
  - Verificar se um número é múltiplo de 5 ou 3
  - Verificar se um triângulo é equilátero, isósceles ou escaleno
  - Verificar o conceito de um aluno baseado na nota

- **Desenhos Geométricos:**
  - Desenhar quadrados completos e vazados com base em um número fornecido pelo usuário

- **Outras Funcionalidades:**
  - Adivinhação de números
  - Cálculo de preço de produtos
  - Verificar se um dia é útil ou fim de semana
  - Verificação de faixa etária
  - Verificação de impostos a pagar com base no salário

## Como Executar

Para executar o projeto, siga os seguintes passos:

1. Clone o repositório:
```bash
 git clone https://github.com/seu-usuario/projetoFinal_PE.git
```
   
2. Navegue até o diretório do projeto:
```bash
cd projetoFinal_PE
```

3. Compile o código:
```bash
gcc -o programa projetoFinalDavi.c
```

4. Execute o programa:
```bash
./exercicios
```
